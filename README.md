## What is this?

A free, growing library of **300+ ML and AI exercises** organised by skill pillar — from data cleaning basics to production RAG pipelines, LLM fine-tuning, multi-agent systems, and deployment.

Each task includes:
- ✅ A real-world problem statement
- ✅ What you will learn
- ✅ Starter code with deliberate gaps (the exercise)
- ✅ Full solution with line-by-line explanation
- ✅ Common mistakes to avoid
- ✅ Related tasks to follow next

**Built for:** ML beginners levelling up · AI engineers building portfolios · Teams upskilling for 2026
